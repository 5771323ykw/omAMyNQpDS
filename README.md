# 项目简介

## 核心功能
- 试题库管理
- 试卷组卷
- 考试管理

## 适用场景
- 教育机构
- 在线学习平台
- 自学爱好者

## 技术特点
- 采用SpringBoot框架，实现快速开发
- 前后端分离，使用Vue技术
- 支持在线题库管理，方便教师和学生使用
- 支持MySQL数据库，保证数据安全性
- 提供详细的文档和视频教程，方便用户快速上手
